#PyChat

Un programme de chat ecrit en python 2.7 qui contient un script serveur et un script client.
De multiples connections client TCP sont possibles.
Les message sont chiffrés à l'aide d'un algorythme RSA.

##Utilisation

Dans un terminal lancez les serveur avec 
``` bash
python server.py
```

Dans une autre fÃªnetre lancez le client avec 
````bash
python client localhost 5000
```
